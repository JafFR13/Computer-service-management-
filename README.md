## 🛠️ Tech Stack & Architecture

- **Language:** Java (JDK 8 or higher)
- **IDE:** NetBeans IDE
- **GUI Framework:** Java Swing / AWT
- **Database:** MySQL
- **Database Driver:** JDBC (MySQL Connector/J)
- **Architecture:** Object-Oriented Programming (OOP)

## 🚀 Key Features

- **Full CRUD Operations:** Seamlessly Create, Read, Update, and Delete service logs and hardware data.
- **Real-Time Search & Filter:** Quickly find specific repair logs based on Client Name, Device Serial Number, or Repair Status.
- **Relational Database Integration:** Structured MySQL database to ensure data integrity between clients, devices, and service histories.
